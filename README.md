# localcbt
> 나만의 단답형 문제 은행

## 개요
- 기획: 간단한 단답형 문제풀이가 가능한 사이트
- 기술: java/spring boot, mysql, bootstrap/css, spring security
- 기능: 문제 등록 및 풀이, 랜덤 문제 풀기
