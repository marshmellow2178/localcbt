# localcbt
> 나만의 단답형 문제 은행

## 개요
간단한 단답형 문제풀이가 가능한 사이트

## 기술 스택
- 백엔드: java 17, spring boot 3, spring security 
- 데이터: mysql
- 프론트엔드: bootstrap/css, thymeleaf

## 주요 기능
- 문제 등록(Create)
- 전체 문제 조회(List, 페이징)
- 랜덤 문제 풀이(Random)
- 개별 문제 상세(조회)
- 정답 제출 및 검증(Answer Check)
- 사용자 인증(Login/Logout)
